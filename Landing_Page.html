HTML

Hyper Text Markup Language (HTML)
is a markup Language used to
structure and display content on
web pages. HTML is a set of codes that tell
web browsers how to display a page's
words and images.

CSS

Cascading Style Sheet (CSS) is a
style sheet language used for specifying
the presentation and styling the
of the Document writtten in a
markup language such as HTML or XML.

Java Script

JavaScript is a programming language
that adds interactivity to web pages.
It is used to create dynamic and interactive web pages,
including animated graphics, content updates,
and more.Javascript is a key part
of web development along with HTML and CSS.
